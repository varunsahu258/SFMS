SFMS Offline Help

Getting Started

Sign in with your assigned account. Administrators must complete the setup wizard on first use. Use Settings to maintain school details, the active academic year, security timeout, appearance and backup schedule.

Collections

Use Main Collection for regular receipts, Small Collection for half-page receipts, and the exemption or advance screens for those workflows. Verify the student, fee heads, mode and amount before saving.

Reports

The Reports center generates daily, monthly, dues, defaulter, year-to-date, audit, collection, discount and void reports. Generated files are stored in the reports folder.

Backups

Create a backup before major data operations. Encryption requires the master backup password. Always inspect the preview before restoring. Archives never clear the live database.

Security

Never edit payment, receipt, hash or audit records outside SFMS. Use Void Payment for corrections. Administrators can inspect tamper alerts in the audit log and manage account locks in User Management.
